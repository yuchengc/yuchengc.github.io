Review website : https://events.umich.edu/

My role : Designer


Make sure there is enough contrast between text and its background color

Yes

Overall, this website applies good contrast between text and the background. It uses the light font color with bark background and dark font color with the light background. Users can easily read the content. But it uses blue-green text with a dark-grey background at some part. This combination is not clear enough for users.

Don't indicate important information using color alone

Yes

Overall, this site uses color to highlight the title and important information. But it doesn't use other ways, such as underline and larger size, to emphasize the content. Also, it uses the bold style for most of the test. This design will also reduce the readability of the important information.

Pair values of colors together (not only hues) to increase contrast

Yes

This site does a great job in this part. it use complementary color to enhance the contrast

Don't rely on sensory characteristics as the sole indicator for understanding and operating content

Yes

This site uses some squares and block to construct the layout and navigates users to click the blocks for further information. It doesn't rely on sensory characteristics.

Design focus states to help users navigate and understand where they are

Yes

This site allows users to use "tab" to focus and select items. But, in the event list, this website doesn't use the hover effect. Thus, users might not notice where they are easily.

Help users understand inputs and help them avoid and correct mistakes

Yes

For its search bar, it provides a hint, "search" text, for users and educates users that they could type something here. After users type some words, the web will provide auto-match content for your to speed up the search process. It is a good design and is efficient.

Write good alt text for your images

Yes

In this site, images has alt text.

If an experience cannot be made accessible, create another route for users to get that information

Yes

It provides an auto-play slideshow at the top of the page. For the slideshow, when the content is hidden, users still can use the right-side menu to switch and access the content.

Be as consistent and clear as possible in layout and copy

Yes

For the events list, it provides a format for all event post. However, because events' title is different, the spaces for titles have various size. This design might make the screen messy. But it still keeps each event area equal.
